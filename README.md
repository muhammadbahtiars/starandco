# Star & Co - The Modern Pastoral

Website landing page untuk Star & Co, sebuah perusahaan dairy premium dengan konsep "The Modern Pastoral".

## 📁 Struktur Project

```
startandco/
├── index.html              # Landing page utama
├── our-farms.html          # Halaman Our Farms
├── products.html           # Halaman Products
├── products-premium-1.html # Detail produk premium 1
├── products-premium-2.html # Detail produk premium 2
├── products-premium-3.html # Detail produk premium 3
├── about.html              # Halaman About Us
├── contact.html            # Halaman Contact Us
├── DESIGN.md               # Design system documentation
├── README.md               # File ini
└── assets/
    └── screenshots/        # Screenshot desain dari Stitch
```

## 🎨 Design System

Design menggunakan konsep **"The Modern Pastoral"** - memadukan kesan organik dari peternakan dengan presisi tinggi dari pengolahan dairy premium.

### Warna Utama
- **Primary**: `#154212` (Deep Green)
- **Surface**: `#f9f9ff` (Milk White)
- **On Surface**: `#101b30` (Dark Blue)

### Typography
- **Headlines**: Manrope
- **Body**: Plus Jakarta Sans

### Prinsip Desain
- No-Line Rule (tanpa border 1px)
- Glassmorphism untuk elemen floating
- Generous whitespace
- Soft corners (rounded-xl, rounded-full)

## 🚀 Cara Menjalankan

Cukup buka file HTML di browser atau gunakan live server:

```bash
# Menggunakan Python
python -m http.server 8000

# Menggunakan Node.js
npx serve .
```
